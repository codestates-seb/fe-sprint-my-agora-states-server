# Getting Started

**Requirements**
* Node.js
* NPM

**Installation**

```
git clone https://github.com/dongbin420/fe-sprint-my-agora-states-server
npm install
```

**Run**

* Server
```
npm start
```

* Client

  바닐라 자바스크립트 프로젝트로, Html파일을 실행.
